 # Приложение: Random Quote Generator

**Автор:** [Мясоедова Анастасия]

## Описание
Программа представляет собой графический интерфейс (GUI) для генерации случайных вдохновляющих цитат. 
Пользователь может получать цитату из базы данных, фильтровать их по определенной теме или автору, 
а также добавлять свои собственные высказывания.

## Основные функции
- **Генерация:** Выбор случайной цитаты с учетом фильтров.
- **Фильтрация:** Выбор категории (Тема) или конкретного автора.
- **История:** Сохранение последних просмотренных цитат в списке.
- **Хранение данных:** Все цитаты и история сохраняются в файл `quotes_data.json` и автоматически загружаются при старте.
- **Валидация:** Проверка на пустые поля при добавлении новой цитаты.

## Инструкция по использованию
1. Запустите файл `main.py`.
2. Выберите фильтры в верхней части окна (по желанию).
3. Нажмите "Сгенерировать цитату".
4. Чтобы добавить свою цитату, введите текст и автора в нижние поля и нажмите "Добавить".

## Тестирование
- Попытка добавить цитату с пустым полем "Текст" вызовет окно ошибки.
- Если выбрать автора, у которого нет цитат в выбранной теме, программа выдаст предупреждение.
